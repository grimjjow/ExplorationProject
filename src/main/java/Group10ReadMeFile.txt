To run the code go to GUI package class Gui and run it.

If there are issues with reading the map file, go to World package and then into Maps package, there you should
take test_3.map or test_4.map

Guard - Boltzmann
Intruder - To Target

The Pathfinding package and Genetic Algorithm package includes not yet integrated code, but it represents
our ideas for the next phase in terms of structure.

The package Mode includes ideas for implementation of pursue/evasion logic(not yet integrated in to the main code)
